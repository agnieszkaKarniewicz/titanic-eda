# Titanic EDA

## Project Overview
Exploratory Data Analysis of the Titanic dataset to identify key survival factors.

## Dataset
The dataset contains passenger information including age, gender, ticket class, and survival status.

## Key Insights
- Gender was the strongest survival predictor.
- First-class passengers had significantly higher survival rates.
- Children had better survival chances.
- Ticket fare correlates with survival probability.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
